# WASM ecosystem

This org maintains some interesting infra about the wasm ecosystem.

- [E2E performance monitor](https://wasm-ecosystem.github.io/performance-monitor/) ([source code](https://github.com/wasm-ecosystem/performance-monitor))
- [unittest framework for AssemblyScript](https://github.com/wasm-ecosystem/assemblyscript-unittest-framework)
- [next generation assemblyscript compiler](https://github.com/wasm-ecosystem/warpo)
- [lightweight wasm aot compiler](https://github.com/wasm-ecosystem/wasm-compiler)
